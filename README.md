# GoCloudBlog

### Installation

`go` package install

```shell script
$ go get -u github.com/gin-gonic/gin
$ go get github.com/satori/go.uuid
$ go get -u github.com/go-sql-driver/mysql
$ go get github.com/go-xorm/xorm
```

`ant design pro` package install

```shell script
$ cd web
$ yarn isntall
```
